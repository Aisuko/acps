Debiasing Large Language Models via Adaptive Causal Prompting with Sketch-of-Thought

ACPS is an adaptive causal prompting framework that selects between standard and conditional front-door adjustment and pairs it with concise Sketch-of-Thought mediators for robust, cost-efficient reasoning.

Read the paper GitHub repo Demo video (soon)
Keywords: LLMs Prompting Causality Sketch-of-Thought

Authors: Bowen Li, Ziqi Xu, Jing Ren, Renqiang Luo, Xikun Zhang, Xiuzhen Zhang, Yongli Ren, Feng Xia

TL;DR: Adaptive causal routing + Sketch-of-Thought mediators deliver robust reasoning with fewer tokens.

Quickstart (notebooks)

  1. Open a Kaggle notebook (Python 3.11.13, GPU recommended).
  2. Install dependencies: pip install -r requirements.txt (the list is heavy; trim if you only need selected notebooks).
  3. Run acps.ipynb for the full pipeline, or the task-specific notebook under acps/.
  4. Use sots_datasets/ notebooks to regenerate Sketch-of-Thought mediators.
